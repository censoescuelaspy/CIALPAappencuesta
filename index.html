Arquitectura del proyecto

Mapa funcional de CIALPA: herramientas, datos, almacenamiento, sincronizacion, evidencias y seguridad operativa.

1

Campo

Encuestador en tablet, celular o notebook. Levanta escuela, bloques, pisos, aulas, sanitarios, exteriores, fotos y tiempos.

Android, iPadOS, escritorio
2

PWA CIALPA

Aplicacion web estatica publicada en GitHub Pages, con Service Worker para cache offline y actualizacion de version.

HTML, CSS, JS, Cache API
3

Registro arquitectonico

Motor MEC, plano por bloque, editor de ambientes, sanitarios, exteriores, base de calles, PDF, DXF, SVG, PNG y JSON.

Canvas, Leaflet, OSM HOT
4

Persistencia local

Autoguardado del borrador y cola offline. Permite seguir cargando aunque la conexion sea intermitente.

localStorage, IndexedDB
5

Sincronizacion

La capa API envia sesiones, escuelas, incidencias, evidencias y estados al backend Google Apps Script.

HTTPS, GAS Web App
6

Repositorios

Datos tabulares en Google Sheets, evidencias en Google Drive y exportaciones tecnicas para gabinete.

Sheets, Drive, anexos
7

Gestion y decision

Mapa, planificacion, estadisticas, auditoria, jornadas, tiempos por escuela y futura capa BigQuery para escala nacional.

Herramientas vinculadas

  • GitHub Pages: publicacion de la app web.
  • Service Worker: app shell offline y cache de version.
  • Google Apps Script: API de login, escuelas, sesiones, incidencias y evidencias.
  • Google Sheets: repositorio tabular operativo.
  • Google Drive: repositorio de fotos y anexos.
  • Leaflet + OSM: mapa general y base de calles del plano.
  • Canvas/SVG/DXF/PDF: produccion tecnica del plano escolar.

Seguridad y controles

  • Acceso por usuario, contrasena y token de sesion.
  • Expiracion local de sesion despues de 8 horas.
  • Roles: encuestador, supervisor y admin.
  • Lista blanca adicional para administradores reales.
  • HTTPS en GitHub Pages y Google Apps Script.
  • La UI oculta funciones, pero el backend debe validar permisos.
  • Los datos locales del navegador dependen de la seguridad del dispositivo.

Escalamiento recomendado para 6000 escuelas

Ahora GitHub Pages + Apps Script + Sheets/Drive + cache offline.
Siguiente Sincronizacion robusta de paquetes CIALPA, bitacora por entidad y tablero de tiempos.
Escala nacional BigQuery como repositorio analitico, Drive/Cloud Storage para fotos, politicas de retencion y backups.
Gobernanza Catalogo de datos, trazabilidad por usuario, version de formulario, auditoria y control de calidad.
